How does it work?

1. Select message

Select the agent that will solve the problem, the priority and a descriptive text.

2. Creating ticket

The ticket will be created and you will be able to view it from MelonHelp, answer any questions and know the status of the ticket.

3. Send email

When the ticket is closed, both the user and the agent will receive an evidence email with the correct answers.

You willl love MelonHelp.

Our ticket system will allow you to see clearly and concisely the status of your ticket, its priority and when it is closed if the user liked or disliked your answers with the CSAT emoticon system.

An email will be sent to you if you have been assigned a ticket.

An evidence email will be sent to the ticket participants as evidence when the ticket is closed.

You will be able to get all the metrics in a simple way and in a few clicks.
